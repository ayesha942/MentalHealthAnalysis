
## About This Project

This project focuses on analyzing a mental health dataset obtained from Kaggle. The aim is to explore how factors such as age, gender, education, and other demographic characteristics relate to depression severity and treatment outcomes. The dataset was cleaned and processed using Python, followed by exploratory data analysis and visualizations to identify meaningful patterns and trends.

Through this project, I practiced data cleaning, handling missing values, feature engineering, and data visualization using Pandas, NumPy, Matplotlib, and Seaborn. The analysis also examines changes in MADRS depression scores before and after treatment to better understand treatment effectiveness and participant improvement.
